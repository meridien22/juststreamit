# JustStreamIt
L’association JustStreamIt est connue pour ses newsletters de classement de films.

Pour lancer son nouveau site internet, ouvrez le fichier index.html dans votre navigateur préféré (Chrome, Firefox, Edge ou Safari). Vous pourrez alors découvrir quels films sont les mieux notés pour chaque catégorie et accéder à de nombreux détails les concernant.

Le site est validé conforme au niveau HTML et CSS selon les standards W3C.
